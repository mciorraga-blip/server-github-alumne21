# server-github-alumne21

Este proyecto consiste en montar un servidor Linux en la nube utilizando GitHub Codespaces.

El trabajo incluye:
  - Acceso remoto mediante SSH
  -  Instalación y configuración del servidor Apache
  -  Una web pública accesible
  -  Una zona privada protegida con autenticación básica

Dentro del repositorio se encuentra la estructura inicial y los archivos necesarios para poner el proyecto en funcionamiento.
